Industrial Robot Arm

Palletizing using PLC and an ABB robot

Project Description

The goal of this project was to pick and place metal blocks using an ABB industrial robot. The robot and PLC code was programmed using RobotStudio, allowing control of the end effector trajectories and actuation of the gripper. Later work involved stacking small metal cubes dropped from a conveyer belt.
